# Snake_Water_Gun
This is the Python CLI based snake_water_gun game where the user can play this game with the computer.
The game similarly work like a Rock,Paper,Scissor.

Snake drinks water and Snake wins
Water sinks Gun and Water win
Gun kills Snake and Gun wins 

# instructions to play:

1 Run the `main.py` file using Python.  
2 The program will ask you to choose one of the following: `snake`, `water`, or `gun`.  
3 The computer will also make a random choice.  
4 The winner of each round will be displayed. 

Note: Install The Python.
